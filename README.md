# Municipalidad de Coltauco - Rediseño Web

Este es un proyecto breve enfocado en la maquetación y modernización de la interfaz del portal web de la **Municipalidad de Coltauco**. El objetivo principal es mejorar la experiencia del usuario, hacer la página completamente responsiva y limpiar la estructura del código.

## Demos en Vivo

Puedes explorar y comparar ambas versiones directamente desde tu navegador:

* **Nueva Versión (Rediseñada):** [Ver el nuevo portal aquí](https://eljuliodev.github.io/Municipalidad-Coltauco/)
* **Versión Antigua (Original):** [Ver el diseño clásico aquí](https://eljuliodev.github.io/Municipalidad-Coltauco/Coltauco.html)

## Cambios Principales

* **Diseño Responsivo:** Adaptación para una visualización perfecta en dispositivos móviles, tablets y escritorio.
* **Modernización Visual:** Interfaz más limpia, accesible y fácil de navegar para los ciudadanos.
* **Optimización de Código:** Separación de estilos y scripts en archivos independientes (`index_style.css` e `index_script.js`) para un mejor rendimiento.
